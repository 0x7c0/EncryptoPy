EncryptoPy

encrypt / decrypt your file with python3!

type of encryption/hash/checksum:

* aes      128, 192, 256
* des
* base     16, 32, 64
* xor
* hash     dsa, md5, ripmed160, sha224, sha256, sha384, sha512
* hmac
* crc      crc32, adler32

future implementation:

* multiple file input
